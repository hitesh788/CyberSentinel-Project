# 🇮🇳 CyberSentinel – Real-Time Cyber Threat Intelligence Dashboard

## 📌 Title

**Developing a Tool to Provide Real-Time Feeds of Cyber Incidents Pertaining to Indian Cyber Space**

---

## 🚀 Overview

CyberSentinel is a **real-time cyber threat intelligence platform** designed to monitor, analyze, and visualize cyber incidents affecting **Indian cyberspace**.

The system aggregates threat data from multiple open-source intelligence feeds and presents it through an **interactive dashboard**, enabling better **threat assessment, situational awareness, and decision-making**.

---

## 🎯 Problem Statement

To protect **Critical Information Infrastructure (CII)** of the nation, it is essential to have **real-time visibility into cyber threats and incidents** specific to Indian cyberspace.

Currently, cyber incident data is scattered across multiple platforms such as:

* Security feeds
* Forums
* Paste sites
* Social media
* Threat intelligence platforms

This project aims to **collect, process, and visualize** such data in a unified system.

---

## 💡 Solution

CyberSentinel provides a **framework for collecting and analyzing cyber threat data** using:

### 1️⃣ Threat Data Aggregation

* Collects data from:

  * OpenPhish (Phishing feeds)
  * CISA (Known exploited vulnerabilities)
  * URLHaus (Malware URLs)
* Supports integration of additional platforms (Reddit, ThreatFox, etc.)

---

### 2️⃣ Intelligent Threat Processing

* Applies **custom risk scoring algorithm** based on:

  * Attack type
  * Source reliability
  * Severity classification
* Simulates **real-time detection behavior**

---

### 3️⃣ Structured Data Framework

* Organizes incidents into:

  * Sector-wise classification
  * Attack types
  * Severity levels
* Maintains a structured dataset for analysis

---

### 4️⃣ Visualization & Insights

* Interactive dashboard featuring:

  * 📊 Sector-based attack charts
  * 📈 Attack type distribution
  * 🗺️ India-based cyber threat map
  * 📡 Live radar scanner
  * 📰 Cybersecurity news feed
  * 📋 Incident feed table

---

### 5️⃣ Actionable Intelligence

* Generates:

  * Risk scores
  * Threat insights
  * Security recommendations

---

## 🏗️ Project Architecture

```
CyberSentinel/
```
